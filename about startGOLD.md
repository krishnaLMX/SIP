startGOLD is a fintech-based digital gold and silver investment platform built using Flutter, designed to provide secure, real-time precious metal purchasing and savings services.

The application enables users to:

* Buy 24K digital gold and silver instantly
* Invest using amount-based or gram-based purchases
* View live market rates with real-time updates
* Complete secure KYC verification
* Make payments through UPI, cards, net banking, and wallets using Cashfree
* Track holdings and transaction history
* Manage withdrawals and redemptions
* Access fintech-grade security protections

The platform follows enterprise-level security architecture including:

* RSA-OAEP-SHA256 payload encryption
* OWASP MASVS-based mobile security
* Root/jailbreak detection
* Session protection
* Secure payment verification
* Runtime protection layers

Technically, the app is built with:

* Flutter + Riverpod architecture
* WebSocket-based live rate streaming
* Centralized API interceptor
* Secure fintech payment lifecycle
* Dynamic KYC and investment workflows

The overall goal of startGOLD is to provide a seamless, secure, and modern digital gold investment experience similar to enterprise fintech and wealth management platforms.
